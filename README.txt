Da caricare su GitHub:
- index.html
- foto-profilo.jpg
- cartella articoli/

Passaggi:
1. Estrarre questo zip.
2. Nel repository GitHub cliccare Add file > Upload files.
3. Trascinare index.html, foto-profilo.jpg e la cartella articoli.
4. Cliccare Commit changes.
5. Attendere 1-3 minuti.
6. Aprire https://domenicobritti-rgb.github.io e aggiornare con Ctrl+F5.

Le tre pagine articolo sono bozze iniziali modificabili.
